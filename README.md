# 国家专利局：实时对抗版 🛡️

## 项目简介
这是一个基于 Node.js + Express + Socket.io 的实时课堂互动游戏，用于教学技术设计的 12 项基本原则。学生通过发明"奇葩专利"和审查他人专利来学习设计原则，并通过群众陪审团系统进行申诉仲裁。

## 技术栈
- **后端**: Node.js + Express + Socket.io
- **前端**: HTML5 + CSS3 + JavaScript
- **数据存储**: 内存存储（适合课堂使用）

## 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动服务器
```bash
node server.js
```
或
```bash
npm start
```

### 3. 访问地址
- **学生端**: `http://[服务器IP]:3000`
- **教师端**: `http://[服务器IP]:3000/admin`

## 游戏流程

### 第一阶段：大厅 (Lobby)
学生输入昵称加入游戏，等待教师开始。

### 第二阶段：资格考 (Qualification)
- **题目数量**: 8 道案例题
- **通过标准**: 答对 6 题或以上（75%）
- **自动进入**: 当 3 名学生通过资格考后，自动进入大乱斗（可在教师端关闭）

### 第三阶段：大乱斗 (Battle)
1. **练手案例**: 进入大乱斗后，每位学生会收到一个系统示例案例进行练习
2. **发明锻造炉**: 学生创造"奇葩发明"并设定违背的原则
3. **审查流水线**: 审查其他学生的发明，找出违背的原则
4. **积分机制**:
   - 审查正确：+10 分
   - 审查错误：-5 分，发明人 +15 分

### 第四阶段：申诉仲裁 (Appeals)
- **申诉条件**: 审查失败的学生可申诉（扣 2 分保证金）
- **群众陪审团**: 系统召集其他合格学生投票（30 秒）
- **陪审奖惩**: 投对 +3 分，投错 -2 分
- **教师裁决**: 可执行民意或强制改判
- **申诉成功**: 审查官 +22 分，发明人 -35 分

### 第五阶段：荣耀结算 (Result)
显示两个排行榜：
- 🏆 **火眼金睛榜**（审查官排行）
- 🤡 **法外狂徒榜**（发明家排行）

## 设计原则（12 项）
1. 科学性原则
2. 创新性原则
3. 安全性原则
4. 实用性原则
5. 经济性原则
6. 可靠性和耐用性原则
7. 标准化原则
8. 工程心理学和生理学原则
9. 时效性原则
10. 最优化原则
11. 法律、道德规范原则
12. 可持续发展原则

## 示例案例
- **倒刺型防抖圆珠笔**: 违背安全性、工程心理学
- **三角形接口充电线**: 违背标准化、经济性
- **全自动课堂代聊机器人**: 违背法律道德
- **一次性纸质雨衣**: 违背实用性、可靠性

## 教师控制功能
- 开始/结束各阶段
- 自动大乱斗开关（默认启用）
- 查看实时排行榜
- 处理申诉仲裁
- 重置游戏

## 配置说明
- **默认端口**: 3000
- **资格考通过标准**: 6/8 题
- **自动大乱斗触发**: 3 人通过
- **陪审团投票时间**: 30 秒
- **申诉保证金**: 2 分
