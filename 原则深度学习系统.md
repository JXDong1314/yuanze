# 设计原则深度学习系统

## 🎯 核心教学目标

让学生深刻理解：
1. **原则之间的因果关系**：违背A原则往往导致违背B原则
2. **原则之间的矛盾关系**：满足A原则可能牺牲B原则
3. **原则的优先级**：某些原则比其他原则更重要
4. **原则的综合应用**：实际设计需要平衡多个原则

### 方案1：原则链识别系统 ⭐⭐⭐⭐⭐

#### 概念
学生不仅要识别违背的原则，还要理解原则之间的因果链。

#### 实现
```javascript
// 在审查结果中显示原则链
if (isExactMatch) {
    const chain = detectPrincipleChain(selectedAnswers);
    if (chain) {
        bonus += 5;
        msg += `\n🔗 识别出原则链：${chain.description}`;
    }
}

function detectPrincipleChain(principles) {
    // 检测因果链
    const chains = [
        {
            pattern: ['1', '4'],
            description: '科学性→实用性（不科学导致不实用）',
            bonus: 5
        },
        {
            pattern: ['3', '8'],
            description: '安全性→工程心理学（不安全影响使用体验）',
            bonus: 5
        },
        {
            pattern: ['5', '12'],
            description: '经济性→可持续性（不经济导致不可持续）',
            bonus: 5
        }
    ];
    
    for (let chain of chains) {
        if (chain.pattern.every(p => principles.includes(p))) {
            return chain;
        }
    }
    return null;
}
```

#### 教学价值
- 培养因果思维
- 理解原则之间的逻辑关系
- 从单点思维到系统思维

### 方案5：原则学习进度追踪 ⭐⭐⭐⭐

#### 概念
追踪每个学生对各个原则的掌握程度。

#### 实现
```javascript
// 用户数据中添加
users[socket.id].principleStats = {
    '1': { correct: 0, total: 0, accuracy: 0 },
    '2': { correct: 0, total: 0, accuracy: 0 },
    // ... 12个原则
};

// 每次审查后更新
function updatePrincipleStats(userId, selectedPrinciples, truePrinciples) {
    truePrinciples.forEach(p => {
        users[userId].principleStats[p].total++;
        if (selectedPrinciples.includes(p)) {
            users[userId].principleStats[p].correct++;
        }
        users[userId].principleStats[p].accuracy = 
            users[userId].principleStats[p].correct / 
            users[userId].principleStats[p].total;
    });
}
```

function updateRelationHints() {
    const selected = getSelectedPrinciples();
    const hints = [];
    
    // 检测因果链
    const chains = detectChains(selected);
    if (chains.length > 0) {
        document.getElementById('chain-detection').style.display = 'block';
        hints.push(`<strong>因果链：</strong>${chains[0].description}`);
    }
    
    // 检测矛盾
    const conflicts = detectConflicts(selected);
    if (conflicts.length > 0) {
        hints.push(`<strong>⚡ 矛盾关系：</strong>${conflicts[0].description}`);
    }
    
    // 显示相关原则
    selected.forEach(id => {
        const related = getRelatedPrinciples(id);
        if (related.length > 0) {
            hints.push(`<strong>${id}号原则</strong>常与 ${related.join(', ')} 一起出现`);
        }
    });
    
    document.getElementById('relation-hints-content').innerHTML = 
        hints.length > 0 ? hints.join('<br>') : '选择原则后显示关联提示';
}

### 实现2：课后分析报告

```javascript
// 生成个人学习报告
function generateLearningReport(userId) {
    const user = users[userId];
    const stats = user.principleStats;
    
    // 计算各项指标
    const totalReviews = Object.values(stats).reduce((sum, s) => sum + s.total, 0);
    const avgAccuracy = Object.values(stats).reduce((sum, s) => sum + s.accuracy, 0) / 12;
    
    // 找出薄弱环节
    const weakPrinciples = Object.entries(stats)
        .filter(([id, s]) => s.accuracy < 0.6 && s.total > 0)
        .map(([id, s]) => ({ id, accuracy: s.accuracy }))
        .sort((a, b) => a.accuracy - b.accuracy);
    
    // 找出擅长领域
    const strongPrinciples = Object.entries(stats)
        .filter(([id, s]) => s.accuracy > 0.8 && s.total > 0)
        .map(([id, s]) => ({ id, accuracy: s.accuracy }))
        .sort((a, b) => b.accuracy - a.accuracy);
    
    return {
        totalReviews,
        avgAccuracy,
        weakPrinciples,
        strongPrinciples,
        relationMastery: calculateRelationMastery(user),
        suggestions: generateSuggestions(weakPrinciples)
    };
}
```

### 活动2：矛盾平衡辩论

**目标**：理解原则之间的权衡

**规则**：
1. 分组辩论
2. 正方：支持原则A
3. 反方：支持原则B
4. 找出最佳平衡点

**示例**：
```
辩题：自动驾驶汽车应该优先考虑创新性还是安全性？

正方（创新性）：
- 技术进步需要大胆尝试
- 长期来看会更安全
- 不创新就会落后

反方（安全性）：
- 人命关天，不容试错
- 需要充分测试
- 安全是底线

平衡点：
- 在封闭场地充分测试
- 渐进式推广
- 建立安全冗余机制
```
