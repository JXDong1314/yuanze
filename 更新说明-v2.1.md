# 国家专利局游戏 v2.1 更新说明

## 🎯 本次更新重点

### 1. 全新资格考试系统

#### 基础考试（必考）
- **改进前**：8道多选题，答对6题（75%）通过
- **改进后**：10道单选题，答对7题（70%）通过

**改进理由：**
- 单选题更简单，降低新手门槛
- 题目数量增加，但难度降低
- 每题只选一个最主要的原则，更符合实际思维

**题目示例：**
```
题目：太阳能手电筒
描述：这款手电筒只能在白天有阳光时使用，晚上无法工作。

选项：
A. 科学性原则
B. 实用性原则 ✓（正确答案）
C. 经济性原则
D. 法律道德原则
```

#### 进阶挑战（可选）
- **全新功能**：通过基础考试后，可选择挑战进阶题
- **题型**：5道多选题
- **奖励**：答对可获得初始分数

**奖励机制：**
| 题目难度 | 原则数量 | 奖励分数 |
|---------|---------|---------|
| 简单 | 1个原则 | +3分 |
| 中等 | 2个原则 | +5分 |
| 困难 | 3个原则 | +8分 |

**最高可获得：** 26分初始分数（全部答对）

## 💡 设计思路

### 问题分析
原来的考试系统存在以下问题：
1. **门槛过高**：多选题对新手不友好
2. **缺少激励**：优秀学生没有额外奖励
3. **体验单一**：所有人都做同样的题目

### 解决方案

#### 1. 分层考试设计
```
基础考试（入门）
    ↓ 通过
获得资格 → 选择是否挑战
    ↓ 是
进阶挑战（提升）
    ↓
获得初始分数优势
```

#### 2. 难度梯度
- **基础考试**：单选题，70%通过率，快速入门
- **进阶挑战**：多选题，可选参与，奖励优秀

#### 3. 心理激励
- 通过基础考试后有成就感
- 进阶挑战是"额外奖励"，不是"必须完成"
- 初始分数优势激励学生挑战自我

## 🎮 游戏流程

### 学生视角

1. **进入资格考试**
   - 看到10道单选题
   - 提示：答对7题即可晋级

2. **提交基础考试**
   - 答对7题或以上 → 通过
   - 答对6题或以下 → 重新尝试

3. **通过后的选择**
   ```
   🎉 恭喜晋级！答对 8/10 题！
   
   你已获得发明家与审查官资格！
   
   [弹窗提示]
   🌟 想要挑战进阶题目获得初始分数吗？
   
   进阶题目更难，但答对可以获得奖励分数！
   
   [确定] 开始挑战    [取消] 直接进入游戏
   ```

4. **进阶挑战（可选）**
   - 5道多选题
   - 每题显示奖励分数
   - 完成后显示获得的总分

5. **进入大乱斗**
   - 带着初始分数优势开始游戏

### 教师视角

- 点击"开始资格考"按钮
- 学生自动收到基础考试题目
- 学生通过后可自主选择是否挑战进阶题
- 3人通过后自动进入大乱斗（可关闭自动）

- 进阶挑战给优秀学生展示机会

- 基础考试保证基本知识掌握
- 进阶挑战考查综合理解能力
- 满足不同水平学生的需求

```javascript
    {

    {
        id: 11,
        type: 'multiple',
        title: "高压电击防盗饭盒",
        desc: "...",

        bonus: 5
    },
    // ... 5题

```

### 客户端

```javascript
// 处理基础考试
socket.on('qualificationQuestions', (data) => {
    if (data.type === 'basic') {
        // 渲染单选题
    } else if (data.type === 'advanced') {
        // 渲染多选题
    }
});

// 通过后询问是否挑战进阶
socket.on('qualificationResult', (res) => {
    if (res.success && res.showAdvanced) {
        const takeAdvanced = confirm("想要挑战进阶题目吗？");
        if (takeAdvanced) {
            socket.emit('requestAdvancedTest');
        }
    }
});
```

## 📝 使用建议

### 课前准备
1. 向学生说明考试分为两部分
2. 强调基础考试是必须的，进阶是可选的
3. 鼓励学生挑战进阶题，但不强制

### 课中引导
1. 观察学生通过率，适时给予鼓励
2. 对挑战进阶题的学生给予肯定
3. 提醒学生进阶题的奖励分数优势

### 课后总结
1. 统计基础考试通过率
2. 统计进阶挑战参与率和得分情况
3. 分析学生对不同原则的掌握程度

---

## ⚠️ 注意事项

1. **基础考试可重复**：学生未通过可以重新尝试
2. **进阶挑战只有一次**：通过基础考试后只能挑战一次
3. **自动进入大乱斗**：3人通过后自动开始（可关闭）
4. **初始分数优势**：进阶题得分会显示在排行榜上
