# 管理端更新说明 v3.0 - 个人进度系统

## 新的学生列表显示

### 显示内容
每个学生显示：
1. 姓名
2. 当前阶段（学习中/考试中/战斗中/需重学）
3. 总分（审查官分+发明家分）
4. 操作按钮

### 阶段标识
- 🎓 学习中 (learning)
- 📝 考试中 (qualification)
- ⚔️ 战斗中 (battle)
- ❌ 需重学 (failed)

### 单个学生操作
- [强制进入学习] - 让学生回到学习阶段
- [强制进入考试] - 让学生进入考试阶段
- [强制进入大乱斗] - 让学生进入战斗（无视积分）

## 新的批量操作按钮

### 批量操作
- [全部进入学习] - 所有学生回到学习阶段
- [全部进入考试] - 所有完成学习的学生进入考试
- [全部进入大乱斗] - 所有通过考试的学生进入战斗
- [结束游戏] - 结束游戏，显示排名
- [重置所有数据] - 清空所有数据

## 实现要点

### 学生列表HTML结构
```html
<div class="student-item">
    <div class="student-info">
        <span class="student-name">张三</span>
        <span class="student-stage">🎓 学习中</span>
        <span class="student-score">总分: 0分</span>
    </div>
    <div class="student-actions">
        <button onclick="forceStage('userId', 'learning')">学习</button>
        <button onclick="forceStage('userId', 'qualification')">考试</button>
        <button onclick="forceStage('userId', 'battle')">大乱斗</button>
    </div>
</div>
```

### 事件发送
```javascript
function forceStage(userId, stage) {
    socket.emit('adminForceStage', { userId, stage });
}

function batchStage(stage, condition) {
    socket.emit('adminBatchStage', { stage, condition });
}
```
