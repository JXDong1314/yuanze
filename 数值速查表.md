## 🎯 典型收益场景

### 场景1：纯审查官（30分钟）
```
进阶考试：+10分
审查10次（7对3错）：+84分 - 9分 = +75分
关联奖励（2次）：+6分
陪审团（5次，3对2错）：+6分 - 4分 = +2分
总计：93分
```

### 场景2：纯发明家（30分钟）
```
进阶考试：+10分
提交5个发明（3个成功）：+36分
审查5次（3对2错）：+36分 - 6分 = +30分
总计：76分
```

### 场景3：平衡型（30分钟）
```

审查7次（5对2错）：+60分 - 6分 = +54分
提交3个发明（2个成功）：+24分
总计：88分
```

### 场景4：申诉流（高风险）
```
进阶考试：+10分
审查5次（3对2错）：+36分 - 6分 = +30分
申诉2次（1成功1失败）：+20分 - 8分 = +12分
总计：52分（风险大，收益不稳定）
```

## 💡 策略建议

### 新手推荐
1. 先做进阶考试（争取10分以上）
2. 主打审查（稳定得分）
3. 尝试1-2个发明（体验玩法）
4. 谨慎申诉（保证金8分较高）

### 进阶玩家
1. 进阶考试全对（16分）
2. 审查+发明平衡（最大化收益）
3. 识别原则关联（额外2-3分）
4. 战略性申诉（有把握时）

### 高手技巧
1. 快速审查（提高效率）
2. 创造迷惑性发明（提高成功率）
3. 精准识别关联（每次+2-3分）
4. 参与陪审团（稳定小额收益）

## ⏱️ 时间管理

### 30分钟游戏建议分配
```
0-5分钟：进阶考试（可选）
5-25分钟：大乱斗主要阶段
  - 审查：每次2-3分钟
  - 发明：每次3-5分钟
  - 陪审：每次1-2分钟
25-30分钟：冲刺阶段
```

### 效率对比
```
审查：3-4分/分钟（最高效）
发明：2-3分/分钟（需要创作）
陪审：1-2分/分钟（等待时间长）
申诉：不确定（高风险）
```

## ⚠️ 常见问题

**Q: 进阶考试必须做吗？**
A: 不必须，但建议做。最多可获得16分优势。

**Q: 申诉值得吗？**
A: 保证金8分较高，只在非常有把握时申诉。

**Q: 审查官和发明家哪个更好？**
A: 平衡发展最优，审查效率高，发明更有趣。

**Q: 关联奖励重要吗？**
A: 重要！每次+2-3分，积少成多。

**Q: 陪审团值得参与吗？**
A: 值得，虽然收益低但风险也低，可以稳定得分。
