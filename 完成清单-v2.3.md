## 🔧 技术实现细节

### 游戏状态枚举
```javascript
gameState = 'lobby' | 'learning' | 'qualification' | 'battle' | 'finished'
```

### 关键事件流
```
教师端 → adminStartLearning → 服务器 → gameState = 'learning' → 广播 → 客户端 → showScreen('learning')
```

### 强制大乱斗流程
```
教师端 → adminForceStartBattle → 服务器 → 
  1. gameState = 'battle'
  2. emit('gameState', 'battle')
  3. emit('forceEnterBattle')
→ 客户端 → 
  1. showScreen('battle')
  2. 关闭所有模态窗口
```
